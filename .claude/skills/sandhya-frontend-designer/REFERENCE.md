# Claude Design Skill – Reference Library

## Craft Checklist

* 8px spacing system
* Predictable padding rhythm
* Visual alignment always intentional
* No floating or ambiguous hierarchy

## Typography Reasoning

Choose type based on:

* Seriousness vs playfulness
* Density of content
* Brand personality

## Motion Timing

| Use case        | Duration  |
| --------------- | --------- |
| Hover           | 100–150ms |
| Enter/exit      | 160–220ms |
| Page transition | 200–300ms |

Easing: ease‑out or cubic‑bezier(0.2, 0, 0, 1)

## Aesthetic Mode Prompts

Use when creativity matters:

* “Editorial minimalism”
* “Warm technical”
* “Playful productivity”
* “Luxury software”

## Enterprise Mode Prompts

Use when reliability matters:

* “Financial SaaS UI”
* “Healthcare dashboard”
* “Admin control panel”

## UX Flow Heuristic

Every screen should answer:

1. Why am I here?
2. What should I do?
3. What happens next?

## Visual Smell Test

Reject if:

* Looks generic
* Lacks hierarchy
* Has arbitrary color
* Feels unfinished
