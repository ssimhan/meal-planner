# Specialized Skills

This directory contains specialized skills for specific use cases. These are used less frequently than core skills but provide powerful capabilities when needed.

## Available Skills

### [frontend-slides](file:///frontend-slides/SKILL.md)
Zero-dependency HTML presentation generator with 10 curated visual styles. Use when creating slide decks or presentations.

### [image-generator](file:///image-generator/SKILL.md)
Gemini-powered branded image generation with customizable brand guidelines. Use when creating branded visuals or marketing assets.

### [c4-architecture](file:///c4-architecture/SKILL.md)
C4 architecture diagrams for system design. Use when documenting system boundaries, container relationships, or architectural changes.

### [composio-integrations](file:///composio-integrations/SKILL.md)
Composio connections to 1000+ external apps (Slack, GitHub, Gmail, etc.). Use when integrating with external services.

## When to Use Specialized Skills

These skills are **not** part of the core development workflow. Use them when:
- Creating presentations or slide decks
- Generating branded images
- Documenting complex architectures
- Integrating with external services

For everyday development, use the core skills in the parent `skills/` directory.
